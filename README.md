# Pizza Sales Analysis (Tableau)

## Project Overview
This project analyzes one year of pizza sales data to understand customer behavior, sales trends, and product performance using Tableau.

## Business Problem
The goal of this analysis is to identify peak ordering hours, best-selling pizzas, revenue patterns, and underperforming products to support data-driven business decisions.

## Tools Used
- Tableau Public
- Excel

## Key Questions
- How many customers order each day?
- What are the peak ordering hours?
- Which pizzas are best sellers?
- Is there any seasonality in sales?
- Which pizzas underperform?

## Deliverables
- Interactive Tableau Dashboard
- Case Study PDF
- Presentation Slides
